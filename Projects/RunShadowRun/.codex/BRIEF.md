## Stack V1 (MVP)
Python 3 + Pygame

## Objectif (MVP)
Un runner jouable (placeholders OK):
- 3 voies + déplacement avant (simulation 2D top/3-voies)
- contrôles: gauche/droite + saut (option dash/glissade plus tard)
- obstacles (collision = échec) + collectibles (pièces)
- score + meilleur score + restart
- UI minimal (menu/run/gameover)

## Contraintes
- Repartir propre: tout le contexte vit dans les fichiers du repo (.codex/* + JOURNAL.md)
- Codex en mode B: propose des PATCH, l'humain valide
- Documentation courte, utile, pas de roman

## Définition de "done" MVP
Une partie démarre en 1 clic, se joue, se termine, et on peut relancer.
