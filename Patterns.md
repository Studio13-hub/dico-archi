# Studio13 — Patterns (Standards)

## Conventions
- Nommage: clair, stable, sans abréviations obscures
- Commits: petits, fréquents, message explicite

## Documentation minimale par projet
- .codex/BRIEF.md : objectifs + contraintes
- .codex/CONTEXT.md : structure + conventions
- .codex/TODO.md : backlog
- .codex/DECISIONS.md : décisions datées
- JOURNAL.md : résumé de session

## Style de journal
- Court
- Daté
- Sections: Fait / Décisions / Prochaine étape / Risques
