# Prochaines etapes (simple)

1. Creer le projet Supabase et activer schema + storage.
2. Mettre les cles dans `config.js`.
3. Creer ton compte via `auth.html`.
4. Activer `is_editor` pour toi.
5. Ouvrir `admin.html` pour gerer les termes.
6. Mettre en place le tableau Google Sheets.
7. Utiliser `scripts/import_terms.py` pour importer.
8. Deployer le site (Vercel ou Netlify).
