# Contributions (formulaire web)

## Cote contributeur
1. Creer un compte dans `auth.html`.
2. Ouvrir `contribuer.html`.
3. Remplir le formulaire et envoyer.
4. Suivre le statut et les commentaires dans "Mes propositions".

## Cote formateur
1. Ouvrir `admin.html`.
2. Charger une proposition, ajuster le contenu si besoin.
3. Ajouter un commentaire.
4. Accepter ou refuser.

Les propositions restent visibles avec leur statut (`draft`, `submitted`, `validated`, `rejected`, `accepted` selon le flux en place).
