# Contributions (formulaire web)

## Cote contributeur
1. Creer un compte dans `auth.html`.
2. Ouvrir `contribuer.html`.
3. Remplir le formulaire et envoyer.

## Cote editeur
1. Ouvrir `admin.html`.
2. Verifier les propositions dans la table `term_submissions` (Supabase).
3. Copier les termes valides vers `terms`.
4. Supprimer la proposition une fois traitee.
