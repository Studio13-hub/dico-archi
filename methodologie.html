Écriture

Comment une fiche se lit bien

Les définitions sont réécrites pour être lisibles, brèves et compréhensibles sans jargon inutile.

1 Nommer clairement

Dire ce qu’est le terme avec des mots clairs.

2 Montrer l’usage

Montrer où le terme apparaît dans le métier.

3 Nettoyer avant sortie

Vérifier la cohérence éditoriale avant publication.

Contrôle

Sources et validation

Sources Normes, pratique, documentation métier

Les fiches s’appuient sur des documents de référence et sur l’usage professionnel.

Validation Apprenti puis formateur

Une fiche est proposée par un apprenti, relue et corrigée par un formateur, puis publiée si elle est prête.

Copyright Réécriture systématique

Les contenus sont reformulés pour éviter le copié-collé de sources externes.

Avant d’écrire Repérer le bon terme

Nommer précisément l’objet, le matériau, le détail ou l’usage visé.

Pendant Écrire pour être utile

Une fiche doit d’abord servir à comprendre vite, avant de chercher l’exhaustivité.

Avant publication Publier quand la lecture tient

Une fiche est prête quand le fond est juste, le vocabulaire net et l’usage lisible.

Une fiche est prête si On comprend vite sans perdre la précision

Le lecteur doit pouvoir nommer, situer et réutiliser le terme sans relire trois fois.

Une fiche n’est pas prête si Le texte reste flou, copié ou trop lourd

Si la lecture fatigue, si la source est reprise telle quelle ou si l’usage reste incertain, on corrige avant sortie.
